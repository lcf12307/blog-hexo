nginx搭建的服务器 报No input file specified错

我把php-fpm重启下就能访问了 然后没过多久又抱No input file specified错

php-fpm在重启下又可以正常访问了

您的服务器不支持 PATH_INFO;

在php.ini文件里面修改：

1、增加一行（这个最重要）

​ fastcgi.impersonate = 1

2、修改两项（解开注释就可以了）

​ cgi.fix_pathinfo=1
​ cgi.force_redirect = 0
